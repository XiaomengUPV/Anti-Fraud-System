  IMPROVEMENT RECOMMENDATIONS
  ──────────────────────────────────────────────────────────

  Diagnosis Mismatch (F1=0.625):
    → High recall (finding many) but low precision (false alarms).
    → Improve: tighten the prompt to require stronger evidence
      before flagging — two-step check: is there ANY valid link?

  Code Substitution (F1=0.690):
    → Very subtle fraud — non-covered code swapped for covered.
      LLM cannot easily detect this without coverage data.
    → Improve: add a coverage lookup table mapping non-covered
      codes to their covered substitutes.
